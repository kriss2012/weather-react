# 🌦️ Weather React

A modern and responsive weather application built with **React.js** that provides real-time weather information for cities around the world. The application fetches live weather data using a weather API and displays essential weather details in a clean, user-friendly interface. 0

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kriss2012/weather-react.git
```

### Navigate to Project

```bash
cd weather-react
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will start on:

```text
http://localhost:3000
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
REACT_APP_WEATHER_API_KEY=YOUR_API_KEY
```

Get your API key from:

```text
https://openweathermap.org/api
```

## 🎯 Usage

1. Open the application.
2. Enter a city name.
3. Click the search button.
4. View live weather information instantly.

The app retrieves current weather data and displays metrics such as temperature, humidity, wind speed, and weather conditions. 2
